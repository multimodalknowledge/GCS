# PyTorch-GCS
GCS loss for 2D facial landmark localization in PyTorch.

## Installation
    $ python 3.6
    $ PyTorch 0.4.1
    $ OpenCV 3.X
    
## Train
    $ python train.py 

mylambda in train.y (line 150) controls the hyper-parameter.
## Test
    $ python netforward.py
   
## Eval
    $ python evaluation.py
